爬虫

对于网络爬虫，这是一个很普遍的东西。在别的语言如java，js中也有。网络世界中的数据杂乱无章，我们需要提取其中的数据并整理成我们需要的格式，这就是网络爬虫的基本概念。

网络爬虫的主要步骤

  1. 通过url获取网页页面信息
  2. 根据正则表达式或者一些html解析方法提取我们需要的内容
  3. 整理，存储提出的内容
  4. 将内容用作它用。

python库的使用

对于返回的网页中中文编码错误问题

《HTTP权威指南》里第16章国际化里提到，如果HTTP响应中Content-Type字段没有指定charset，则默认页面是’ISO-8859-1’编码。这处理英文页面当然没有问题，但是中文页面，就会有乱码了！所以有时候，我们需要查看响应头，如果里面的content-type中没有字符编码，我们可能就需要把返回的字节通过网页中声明的字符编码给解码。

对于爬虫程序中最主要的限制性能的方面主要又3个方面

1.页面解析速度: 对于HTML这种结构化的格式，用正则表达式显然是最慢的，一般都使用第三方库，比如beautifulsoup4、lxml(xpath)、pyquery等。其中lxml是由c语言写的，速度最快。

2.网络IO: 这里主要是网络IO的速度本来就慢，我们可以使用多线程或者协程的方式来加快速度。

3.本地IO: 对于需要写入本地的爬虫程序，我们可以用一些速度快的数据库，比如Redis内存数据库。

处理数据

对于爬取的数据，它肯定是有价值的。比如我们使用该数据来进行分析，或者提供服务，我们又可以使用一些科学数据分析的框架进行数据分析。我们还可以搭建一些微服务框架来给我们的系统提供数据api服务等。
