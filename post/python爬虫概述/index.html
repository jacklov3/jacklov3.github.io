爬虫

对于网络爬虫，这是一个很普遍的东西。在别的语言如java，js中也有。网络世界中的数据杂乱无章，我们需要提取其中的数据并整理成我们需要的格式，这就是网络爬虫的 基本概念。

网络爬虫的主要步骤

  1. 通过url获取网页页面信息
  2. 根据正则表达式或者一些html解析方法提取我们需要的内容
  3. 整理，存储提出的内容
  4. 将内容用作它用。
